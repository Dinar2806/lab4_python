# Library Simulation System

Лабораторная работа №4: Симуляция с пользовательскими коллекциями и псевдослучайной моделью

## Описание

Система управления библиотекой с:
- Пользовательскими коллекциями (BookCollection, IndexDict)
- Сохранением данных в JSON файлы
- Псевдослучайной симуляцией работы
- Наследованием и магическими методами

## Установка

```bash
# Клонирование репозитория
git clone https://github.com/Dinar2806/lab4_python.git
cd library_simulation

pip install -r requirements.txt
```

## Запуск

```bash
python -m src.library_sim.main
```
